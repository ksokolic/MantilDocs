# mantil stage use

Sets default project stage

### USAGE
```
  mantil stage use &lt;stage&gt;
```
### ARGUMENTS
```
  &lt;stage&gt;  Name of the stage which will be default.
```
### GLOBAL OPTIONS
```
      --help       Show command help
      --no-color   Don't use colors in output
```
